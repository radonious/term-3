5. Потоки ввода-вывода
Цель: Ознакомиться с иерархией потоков ввода-вывода.
Научиться пользоваться выводом в потоки. Научиться выводить объекты в двоичном виде в файлы.

Задачи:
- Сделать операторы чтения и записи объекта в поток ввода/вывода.
    --> перегрузить << >> для классов задания чтобы использовать их для записи и чтения в файл и консоль

- Сделать методы для чтения и записи объекта в двоичном виде.
    --> сдеелать методы "string toBin(className obj)" и "className fromBin(string bin)"
        для составления+записи и чтения+расшифровки двоичной формы объекта из файла и консоли

- Вывести стандартным методом объект в двоичном виде в поток.
    --> с помощью << и >> считать и записать двоичный объект в файл и консоль
